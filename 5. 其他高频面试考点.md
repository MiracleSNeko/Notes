### 引用折叠与完美转发

```cpp
#include <bits/stdc++.h>

template <typename T>
// 此处，&& 不再表示右值引用，而是同时可以表示两种引用
// 这里的 t 既可以传入左值引用，也可以传入右值引用
// 什么时候是左，什么时候是右？
// 引用折叠规则： 左侧是入参变量类型，右侧是形式参数类型
//				T  T& -> T&
//				T  T&&(右值) -> T&
//				T& T& -> T&
//				T  T&&(万能) -> T&
//				T& T&&(万能) -> T& && -> T& (& &折叠)
//				T&&（右值） T& -> T&& & -> T& (& &折叠)
//				T&&（右值） T&&（万能） -> T&& && -> T&&（右值）(& &折叠)
void use_reference(T&& t)
{
    // 如果要将引用继续往下传递，不能直接传
    fun(t);                // 具名，传递的是左值引用
    fun(std::forward(t));  // 透传，不改变 t 原有的引用类型
};

template <typename T>
void fun(T&& t); // 万能引用
auto _ = [](auto&& t) {}; // 泛型 lambda 表达式，C++14，万能引用
```
